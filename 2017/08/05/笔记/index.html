浏览器常见兼容性问题

FireFox和IE设置透明度
IE: filter:progid:DXImageTransform.Microsoft.Alpha(style=0,opacity=60)
FireFox: opacity:.6
FireFox和IE对盒模型解释不一致导致相差2px
`例：box.style{
width：100px;
border 1px;
}
IE理解为width=100px
FF则理解为width=100+1*2=102px//算上了边框

解决方法：div{
margin：30px！important;
margin：28px;
}
因为IE不能识别!important这个属性所以会解析成
div{
width:30px;
width:28px;
}
这样就会覆盖前面的定义但是其他浏览器可以解析!important;IE6下margin加倍的问题在IE6下设置了float属性的div其margin会加倍解决方法就是将div变成行内元素`

IE下如何设置最小宽度？
`
.demo{
width：expression(document.body.clientWidth<600?’600px’:’auto’)
}
